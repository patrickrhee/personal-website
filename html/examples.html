Minecraft Plugin

Language Used: Java

This minecraft plugin works inside my minecraft server alongside plugins made by other developers. It is a custom shop plugin which listens for certain in-game events and interacts with the player/client. The purpose of this project was to learn how to use an API in my code (in this case, the minecraft Spigot API), and practice coding for practical purposes (my minecraft server).

Multiplayer Game With Custom Game Engine

Language Used: C++

This is a puzzle and stealth based game in which 2 players must collaborate in real time to complete a level. An agent runs around the level completing tasks and knocking out guards while a hacker assists him with opening sealed doors and collaborating on puzzles. The game is built with a custom game engine built from the ground up. It has custom AI and level generation, and a publicly accessible game server is hosted on google cloud. I worked on the networking team which consisted of 2 other people. The group in total involved 10 people including me all working remotely. Git was used for version control.

Binary Search Tree

Language Used: Java

This program looks for an optimal "Binary Search Tree" given a set of nodes and the number of times each one was accessed. To put it simply, all possible trees are generated, and the most efficient tree is returned by the program. The main purpose of this project was to practice "top-down dynamic programming".
